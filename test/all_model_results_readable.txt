## Remove Duplicates from a List in Java

Here are several approaches to remove duplicates from a list in Java:

### Approach 1: Using LinkedHashSet (Preserves Order)

```java
import java.util.*;

public class RemoveDuplicates {

    public static <T> List<T> removeDuplicates(List<T> list) {
        if (list == null || list.isEmpty()) {
            return new ArrayList<>();
        }
        return new ArrayList<>(new LinkedHashSet<>(list));
    }

    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 2, 4, 3, 5, 1);
        System.out.println("Original: " + numbers);
        System.out.println("Without duplicates: " + removeDuplicates(numbers));
        // Output: Without duplicates: [1, 2, 3, 4, 5]
    }
}
```

### Approach 2: Using Java Streams (Modern Approach)

```java
import java.util.*;
import java.util.stream.*;

public class RemoveDuplicates {

    public static <T> List<T> removeDuplicates(List<T> list) {
        if (list == null || list.isEmpty()) {
            return new ArrayList<>();
        }
        return list.stream()
                   .distinct()
                   .collect(Collectors.toList());
    }

    public static void main(String[] args) {

        System.out.println("Without duplicates: " + removeDuplicates(words));
        // Output: Without duplicates: [apple, banana, cherry]
```

### Approach 3: Manual Iteration (No Extra Libraries)

```java
import java.util.*;

public class RemoveDuplicates {

    public static <T> List<T> removeDuplicates(List<T> list) {
        if (list == null || list.isEmpty()) {
            return new ArrayList<>();
        }

        List<T> result = new ArrayList<>();
        for (T element : list) {
            if (!result.contains(element)) {
                result.add(element);
            }
        }
        return result;
    }

    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(5, 3, 1, 3, 5, 7, 9, 7);
        System.out.println("Original: " + numbers);
        System.out.println("Without duplicates: " + removeDuplicates(numbers));
        // Output: Without duplicates: [5, 3, 1, 7, 9]
    }
}
```

### Approach 4: In-Place Removal (Modifies Original List)

```java
import java.util.*;

public class RemoveDuplicates {

    public static <T> void removeDuplicatesInPlace(List<T> list) {
        if (list == null || list.isEmpty()) {
            return;
        }

        Set<T> seen = new HashSet<>();
        Iterator<T> iterator = list.iterator();

        while (iterator.hasNext()) {
            T element = iterator.next();
            if (!seen.add(element)) {  // add() returns false if element already exists
                iterator.remove();
            }
        }
    }

    public static void main(String[] args) {
        List<Integer> numbers = new ArrayList<>(Arrays.asList(1, 2, 2, 3, 4, 4, 5));
        System.out.println("Before: " + numbers);
        removeDuplicatesInPlace(numbers);
        System.out.println("After: " + numbers);
        // Output: After: [1, 2, 3, 4, 5]
    }
}
```

### Remove Duplicates from a List in Java
#### Using Java 8 Stream API

Here's an example of a function that uses Java 8's Stream API to remove duplicates from a list.

```java
import java.util.List;
import java.util.stream.Collectors;

public class Main {

    /**
     * Removes duplicates from a list.
     *
     * @param list the input list
     * @return a new list with duplicates removed
     */
    public static <T> List<T> removeDuplicates(List<T> list) {
        return list.stream()
                .distinct()
                .collect(Collectors.toList());
    }

    public static void main(String[] args) {
        List<Integer> numbers = List.of(1, 2, 2, 3, 4, 4, 5, 6, 6);
        List<Integer> uniqueNumbers = removeDuplicates(numbers);
        System.out.println("Unique numbers: " + uniqueNumbers);
    }
}
```

#### Using HashSet

If you prefer not to use Java 8's Stream API, you can achieve the same result by using a `HashSet` to store unique elements.

```java
import java.util.*;

public class Main {

    /**
     * Removes duplicates from a list.
     *
     * @param list the input list
     * @return a new list with duplicates removed
     */
    public static <T> List<T> removeDuplicates(List<T> list) {
        Set<T> set = new HashSet<>(list);
        return new ArrayList<>(set);
    }

    public static void main(String[] args) {
        List<Integer> numbers = List.of(1, 2, 2, 3, 4, 4, 5, 6, 6);
        List<Integer> uniqueNumbers = removeDuplicates(numbers);
        System.out.println("Unique numbers: " + uniqueNumbers);
    }
}
```

**Note**: Using a `HashSet` does not preserve the original order of the list. If you need to preserve the order, you can use a `LinkedHashSet` instead.

```java
public static <T> List<T> removeDuplicates(List<T> list) {
    Set<T> set = new LinkedHashSet<>(list);
    return new ArrayList<>(set);
}
```

### Choosing the Right Approach

* Use the Stream API approach when you need to perform additional operations on the list, such as filtering or mapping, as it provides a more functional programming style.
* Use the `HashSet` or `LinkedHashSet` approach when you need a straightforward and efficient way to remove duplicates, especially for large lists.
