# NullPointerException in Java

A `NullPointerException` (NPE) occurs when you try to **use a reference variable that points to no object** (i.e., it's `null`).

---

## Common Causes

### 1. Calling a method on a null object
```java
String str = null;
str.length(); // NullPointerException!
```

### 2. Accessing a field of a null object
```java
Person person = null;
person.name; // NullPointerException!
```

### 3. Uninitialized object reference
```java
String[] arr = new String[5];
arr[0].length(); // arr[0] is null by default!
```

### 4. Method returns null (not handled)
```java
String value = map.get("key"); // returns null if key doesn't exist
value.toUpperCase(); // NullPointerException!
```

### 5. Null passed as a parameter
```java
public void process(String s) {
    s.trim(); // NPE if caller passes null
}
```

---

## How to Fix It

### ✅ Check for null before use
```java
if (str != null) {
    str.length();
}
```

### ✅ Use `Objects.requireNonNull()`
```java
public void process(String s) {
    Objects.requireNonNull(s, "s must not be null");
    s.trim();
}
```

### ✅ Use `Optional` (Java 8+)
```java
Optional<String> value = Optional.ofNullable(map.get("key"));
value.ifPresent(v -> System.out.println(v.toUpperCase()));
```

### ✅ Use ternary or default values
```java
String result = (str != null) ? str : "default";
```

### ✅ Use `getOrDefault()` for Maps
```java
String value = map.getOrDefault("key", "default");
```

---

## Debugging Tips

| Tip | Description |
|-----|-------------|
| **Read the stack trace** | It tells you the exact line number |
| **Java 14+ helpful NPEs** | Shows *which* variable was null |
| **Use a debugger** | Set a breakpoint at the NPE line |
| **Add null checks** | Trace back where null was introduced |

---

## Java 14+ Helpful NPE Messages
```
Cannot invoke "String.length()" because "str" is null
```
*Java now tells you exactly which variable caused the problem.*

---

The key principle: **never assume a reference is non-null** — always validate input and handle potential null return values.
